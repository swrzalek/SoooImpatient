# SooImpantient


#### Options
- Set date in future
- Set a message
- Set a number

#### Result
*App sends everyday at 2:30 PM notification via SMS reminding you how many days are left untill your date *

#### Why?
Because I was so unpantient waiting for trip that I wrote this. Every day me and my friends were informed how long we have to wait.
